What this means

A verified result means the WitnessKey / LOOPtLOOP API confirmed the event metadata and receipt signature it recorded for that authorization event. This page does not verify the private authorization payload itself because WitnessKey never receives that private payload.

This page does not ask you to upload the private payload. If you are comparing a private authorization receipt to a local private input, compute and compare the local sha256: hash without transmitting the private input here.
